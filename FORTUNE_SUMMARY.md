## 🛠️ 技术实现

### 数据库设计
```sql
-- 用户出生日期
profiles.birth_date (DATE)

-- 抽签记录（每日缓存）
fortune_draws (
  user_id,
  birth_date,        -- 抽签时的出生日期
  draw_date,         -- 抽签日期（每日）
  image_url,         -- 运势图片
  fortune_content,   -- 运势内容
  UNIQUE(user_id, draw_date)  -- 每天只能抽一次
)
```

### Edge Function
```typescript
supabase/functions/draw-fortune/index.ts

功能：
1. 验证用户身份
2. 查询出生日期
3. 检查今日是否已抽签（缓存）
4. 调用Coze工作流API
5. 保存结果到数据库
6. 返回运势数据
```

### 前端组件
```typescript
src/components/FortuneDrawDialog.tsx

状态管理：
- isDrawing: 抽签动画
- needBirthDate: 设置出生日期
- fortuneData: 运势结果

三种视图：
1. 抽签动画层
2. 出生日期设置层
3. 结果展示层
```
