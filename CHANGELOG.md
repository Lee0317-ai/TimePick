# 拾光 2.0 更新日志

## 🎉 版本 2.0.0 (2026-01-25)

### ✨ 新增功能

#### 📁 文件夹系统
- 用户可自定义创建文件夹（支持无限层级嵌套）
- 混合展示：文件夹 + 资源同屏显示
- 拖拽资源到文件夹进行组织
- 文件夹统计（子文件夹数量 + 资源数量）
- 面包屑导航支持快速跳转

#### 🏷️ 标签系统
- 为资源添加标签（最多10个）
- 标签云展示（根据使用频率调整大小）
- 标签筛选（支持多标签 AND 查询）
- 标签输入支持快捷键（Enter 添加）
- 标签统计显示

#### 💡 灵感记录
- 快速记录灵感想法
- 🎤 语音输入支持（Web Speech API）
- 位置标记功能
- 灵感列表查看和管理
- 支持编辑和删除

#### 🔍 增强搜索
- 支持搜索标签
- 支持搜索内容描述
- 多维度搜索（名称、标签、备注、URL、内容）
- 搜索结果分组显示
- 搜索结果数量提示

### 🎨 交互优化（新增）

#### 📏 可调整侧边栏
- **拖拽调整**：鼠标拖拽侧边栏边缘即可调整宽度
- **宽度范围**：200px - 500px（默认 280px）
- **持久化**：宽度设置自动保存到浏览器
- **视觉反馈**：拖拽时显示高亮指示器
- **流畅动画**：平滑的拖拽体验

#### ✨ 卡片 Hover 效果增强
- **资源卡片**：
  - 缩放和上浮动画（scale + translateY）
  - 图片缩放效果
  - 标题颜色渐变
  - 桌面端隐藏操作按钮，hover 时显示
  - 移动端始终显示操作按钮
  - 底部操作栏优化
  
- **文件夹卡片**：
  - 3D 提升效果
  - 渐变背景增强
  - 图标动画
  - 操作按钮渐显效果

#### 📱 移动端底部导航（新增）
- **固定底部导航栏**：
  - 分类（打开侧边栏）
  - 搜索（跳转搜索页）
  - 添加资源（中央大按钮）
  - 灵感记录
  - 个人中心
- **悬浮按钮设计**：中央"+"按钮悬浮突出
- **安全区适配**：支持 iOS 底部安全区
- **图标+文字**：清晰的功能标识

#### 🎭 空状态优化
- **精美的空状态组件**：
  - 大图标展示
  - 清晰的标题和描述
  - 渐变背景设计
  - 友好的引导文案
- **多场景支持**：
  - 空文件夹
  - 无资源
  - 搜索无结果
- **行动号召**：提供明确的下一步操作

## 🎯 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **UI 组件**: shadcn/ui + Tailwind CSS
- **后端服务**: Supabase
- **数据库**: PostgreSQL
- **存储**: Supabase Storage
- **认证**: Supabase Auth
- **语音识别**: Web Speech API
- **分析**: Google Analytics + PostHog

---

## 📦 数据库结构

### 主要表
- `profiles` - 用户档案
- `user_roles` - 用户角色
- `folders` - 文件夹（支持嵌套）
- `resources` - 资源（支持标签）
- `inspirations` - 灵感记录
- `sections` - 预设板块
- `modules` - 预设模块
- `search_history` - 搜索历史
