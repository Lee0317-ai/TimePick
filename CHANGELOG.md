# 拾光 2.0 更新日志

## 🎉 版本 2.0.0 (2026-01-25)

### ✨ 新增功能

#### 📁 文件夹系统
- 用户可自定义创建文件夹（支持无限层级嵌套）
- 混合展示：文件夹 + 资源同屏显示
- 拖拽资源到文件夹进行组织
- 文件夹统计（子文件夹数量 + 资源数量）
- 面包屑导航支持快速跳转

#### 🏷️ 标签系统
- 为资源添加标签（最多10个）
- 标签云展示（根据使用频率调整大小）
- 标签筛选（支持多标签 AND 查询）
- 标签输入支持快捷键（Enter 添加）
- 标签统计显示

#### 💡 灵感记录
- 快速记录灵感想法
- 🎤 语音输入支持（Web Speech API）
- 位置标记功能
- 灵感列表查看和管理
- 支持编辑和删除

#### 🔍 增强搜索
- 支持搜索标签
- 支持搜索内容描述
- 多维度搜索（名称、标签、备注、URL、内容）
- 搜索结果分组显示
- 搜索结果数量提示

### 🔧 功能调整

#### 隐藏功能
- ❌ 隐藏视频上传功能
- 视频板块在资源创建时不可选择
- 自动判断文件类型时移除视频类型

#### 优化改进
- 📊 资源卡片显示标签（最多显示3个）
- 🎨 标签徽章设计优化
- 📱 移动端体验优化
- ⚡ 搜索性能优化（客户端过滤）
- 🎯 搜索提示信息优化

### 🎨 设计改进
- 统一的颜色主题
- 精美的卡片设计
- 流畅的动画效果
- 响应式布局优化

### 🐛 Bug 修复
- 修复标签输入问题
- 修复文件夹嵌套显示
- 优化数据库查询性能
- 修复面包屑导航

## 🎯 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **UI 组件**: shadcn/ui + Tailwind CSS
- **后端服务**: Supabase
- **数据库**: PostgreSQL
- **存储**: Supabase Storage
- **认证**: Supabase Auth
- **语音识别**: Web Speech API
- **分析**: Google Analytics + PostHog

---

## 📦 数据库结构

### 主要表
- `profiles` - 用户档案
- `user_roles` - 用户角色
- `folders` - 文件夹（支持嵌套）
- `resources` - 资源（支持标签）
- `inspirations` - 灵感记录
- `sections` - 预设板块
- `modules` - 预设模块
- `search_history` - 搜索历史
