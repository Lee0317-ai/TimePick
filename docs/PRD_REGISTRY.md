# TimePick PRD 总集（台账）

本文件记录 TimePick 项目所有 PRD 文档的最新版本链接。

| 版本 | 标题 | 需求内容（详细摘要） | PRD 链接 |
|------|------|---------------------|----------|
| PRD-001 | TimePick 现有功能基线 | **目标**：记录 TimePick 1.0 版本的所有核心功能，作为基线文档。<br/>**范围**：账户与身份、资源收集（文件上传/URL保存）、组织管理（文件夹/标签/模块视图）、灵感速记、抽签占卜（每日抽签/AI占卜）。<br/>**关键规则**：用户认证、资源类型管理（网页/文档/图片/视频）、文件夹层级结构、标签多维度分类、基于生日的抽签功能。<br/>**边界**：移动端响应式设计、存储配额限制。<br/>**异常**：文件上传失败、网络错误、抽签接口异常处理。<br/>**非目标**：待尝试链接管理（见 PRD-002）。 | [PRD-001-现有功能基线](./prd/PRD-001-现有功能基线.md) |
| PRD-002 | 待尝试链接管理功能 | **目标**：新增待尝试链接管理功能，解决用户"想做的事情太多，无法排优先级"的痛点，支持从微信等渠道批量导入链接并智能排序。<br/>**范围**：快速添加单条链接、批量导入（从微信粘贴）、学习重点与相关性优先级、手动调整优先级、待尝试队列与实践流程、转化为资源或归档。<br/>**关键规则**：三重优先级算法（关键词分析+学习相关性+手动标签）、权重系统（0.5/1.0/2.0）、自定义同义词库、30天自动归档。<br/>**边界**：单次批量导入最多50条、同时允许多个"尝试中"链接。<br/>**异常**：重复链接提示、网页无法访问但仍可保存、优先级计算失败降级处理。<br/>**非目标**：浏览器插件导入（未来版本考虑）。 | [PRD-002-待尝试链接管理](./prd/PRD-002-待尝试链接管理.md) |
