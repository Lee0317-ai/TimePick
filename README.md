# 拾光 (ShiGuang)

一款高效的个人知识与资源管理系统，帮助您收集、整理和检索网页、文档、图片和视频资源。

## 功能特性

- **资源管理** - 收藏、分类、检索各类数字资源
- **灵感速记** - 快速记录想法和灵感
- **待办事项** - 管理待尝试链接和任务
- **AI智能分类** - 自动识别和分类资源
- **全文搜索** - 快速找到需要的内容
- **抽签占卜** - 每日运势和趣味抽签

## 技术栈

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Supabase
- shadcn/ui
